# Call Center Dashboard (Excel)

## Project Overview
This dashboard analyzes call center performance using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting

## KPIs
- Total Calls
- Resolved Calls
- Resolution Rate
- Average Handle Time
- Average Customer Rating
- Number of Agents
- Number of Teams

## Dashboard Features
- Interactive slicers
- Dynamic Pivot Charts
- KPI Cards
- Team Performance Analysis
- Shift Analysis
- City Analysis
- Customer Satisfaction
