# HR Analytics Dashboard (Excel)

## 📌 Project Overview
This project is an interactive HR Analytics Dashboard built in Microsoft Excel using a dataset of 300 employee records.

## 📊 KPIs
- Total Employees
- Average Salary
- Average Experience
- Attrition Rate

## 📈 Dashboard Features
- Department-wise Employee Count
- Gender Distribution
- Performance Rating Analysis
- Location-wise Employees
- Attrition Analysis

## 🛠 Skills Used
- Excel Tables
- Data Cleaning
- Conditional Formatting
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
