# Sales Performance Dashboard (Power BI)

## Project Overview

This interactive Power BI dashboard analyzes sales performance across different regions, product categories, and products.

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Excel

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Profit Margin

## Dashboard Features

- Sales Trend by Month
- Sales by Category
- Profit by Region
- Product Sales Treemap
- Interactive Slicers

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- Relationships
- DAX Measures
- Dashboard Design
- Business Insights
